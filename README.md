# MOD-60 Digital Timer

A digital timer designed using cascaded MOD-60 counters to display time from 00:00 to 59:59 on 7-segment displays.

## Project Overview
The timer utilizes synchronous BCD counters, BCD-to-7-segment decoders, and four 7-segment displays to accurately count and display minutes and seconds. The design was simulated and verified using NI Multisim.

## Features
- Time display from 00:00 to 59:59
- Cascaded MOD-60 counter design
- BCD to 7-segment display interfacing
- Clock synchronization and carry propagation
- NI Multisim simulation

## Technologies Used
- Digital Logic Design
- 74LS160 / 74LS190 Counters
- 74LS47 Decoder
- 7-Segment Displays
- NI Multisim
